# Weekly Research Folders

This directory contains your manually curated articles and reports, organized by week.

## Folder Structure

Create folders with the naming convention: `week-YYYY-MM-DD/`

Example:
```
data/
  week-2024-01-15/
    article-001-coindesk-bitcoin.md
    article-002-theblock-defi.md
    report-003-delphi-ethereum.md
  week-2024-01-22/
    ...
```

## File Naming Convention

Use this format: `[type]-[number]-[source]-[topic].[ext]`

- **type**: `article` or `report`
- **number**: Sequential number (001, 002, 003...)
- **source**: Short identifier (coindesk, theblock, delphi, fintechtakes)
- **topic**: Brief topic description (use hyphens)
- **ext**: File extension (.md, .html, .txt, .pdf)

### Examples

- `article-001-coindesk-bitcoin.md`
- `article-002-theblock-defi.md`
- `report-003-delphi-ethereum.md`
- `article-004-fintechtakes-regulation.html`

## Supported Formats

- **Markdown** (.md) - Recommended
- **HTML** (.html)
- **Plain Text** (.txt)
- **PDF** (.pdf)

## Tips

1. Use Markdown format when possible for best results
2. Include source attribution in the document itself
3. Keep articles focused on single topics when possible
4. The engine will automatically extract content and identify sources
